O que é HTML?

(Voltar)

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa HiperText Markup Language, traduzindo ao português: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Como é uma estrutura básica de um HTML?

(Voltar)

Quando pensamos na estutura básica do HTML, podemos fazer uma comparação com o corpo humano. Da mesma forma, o HTML é dividido em "HEAD" (cabeça) e "BODY" (corpo). Utilizamos as chamadas "TAGS" para fazer as marcações, dessa forma o computador nos apresenta o resultado visual que esperamos. Na tag "HEAD" são adicionadas informações para o funcionamento correto da página. Já na tag "BODY" utilizamos diversas tags, para dar forma ao corpo da página.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Existem tags com a finalidade de criar listas. Essas listas podem ser ordenadas:

  1. Primeiro
  2. Segundo
  3. Terceiro

E listas não ordenadas:

  • Primeiro
  • Segundo
  • Terceiro

Usamos as listas ordenadas quando queremos listar itens de uma forma sequencial, e usamos as listas não ordenadas quando não precisamos listar itens, mas os mesmo não precisam seguir uma sequencia.
